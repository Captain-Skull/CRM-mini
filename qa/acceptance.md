# Acceptance Criteria (Критерии приёмки) — Кейс 3. CRM-мини

**Версия:** 1.0  
**Дата:** 16.02.2026  
**Владелец:** Аналитик (BA/SA) + PL

## Инструкция по использованию
- Каждый критерий должен быть **проверяемым** (Given/When/Then).
- В колонке «Доказательство» после выполнения проставляется ссылка на:
  - Pull Request, где был реализован критерий,
  - или прямой скриншот/видео (в `/docs/release/`),
  - или ссылка на запущенный отчёт/функцию.
- **Вес** указан в скобках для итоговой оценки (сумма = 100%).
